1. **Перемещение между директориями**
``` 
cd <путь_к_директории> 
```
Меняет текущую директорию.

---

2. **Вывод списка файлов в директории**
```
ls
```
Выводит имена файлов и папок в текущей директории.  

---

3. **Вывод списка всех файлов в директории**
```
ls -a  
```
Выводит файлы, включая скрытые (начинающиеся с "."), в текущей директории. Часто используемые флаги: ```-l``` (длинный вывод) и ```-h``` (вывод с размерами файлов).

4. **Создание папки с подпапками**
```
mkdir -p <папка>/<подпапка>/<подпапка подпапки>/...
```
Создает папку и все указанные через слеш до нее.  

---

5. **Внутри папки создать файлик и записать в него что-нибудь**
```
echo "текст" > <имя файла>
```
Создает файл с указанным текстом.  

---

6. **Перемещение файла из одной директории в другую**
```
mv <путь_к_файлу> <путь_к_директории>  
```
Перемещает файл в указанную директорию.  

---

7. **Копирование файла из одной директории в другую**
```
cp <путь_к_файлу> <путь_к_директории>  
```
Копирует файл в указанную директорию. Чтобы скопировать рекурсивно - ```-r```.  

---

8. **Переименование файла**
```
mv <старое_имя> <новое_имя>
```
Переименовывает файл.  

---

9. **Сравнение содержимого файла**
```
diff <файл1> <файл2>
```
Сравнивает содержимое двух файлов, выводит отличия.    

---

10. **Сортировка содержимого файла по возрастанию и убыванию**
```
Возрастание: sort <файл>

Убывание: sort -r <файл>
```  

---

11. **Удаление файлов и папок**
```
rm -r <директория>
```
Удаляет указанную директорию и все её содержимое. rm -rf / - удалит все файлы и директории в корневом каталоге и всех его подкаталогах, включая системные файлы, пользовательские данные и все остальное на диске безвозвратно, не запросив подтверждение
